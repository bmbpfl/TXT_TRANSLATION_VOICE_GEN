# TXT_TRANSLATION_VOICE_GEN

Text/Document/PDF file content -> Translation to target Language -> Voice Generation for the Translated Text

    << Source file can be || Text or Word or PDF || File >>

1. Download python Python 3.14.3 version

			https://www.python.org/downloads/windows/

			search for 3.14.3 version
2. 

	python3 -m venv Virtual_Env
	
	Virtual_Env\Scripts\Activate
	
3.  

	PIP INSTALL streamlit
	
	PIP INSTALL deep_translator
		
	PIP INSTALL docx
	
	PIP INSTALL PyPDF2
	
	PIP INSTALL gTTS	

		   or 
	
		pip install -r requirements.txt

4. 

# STREAMLIT RUN txt_translation_to_voice_generation.py

    this will execute the python code
